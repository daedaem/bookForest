# bookForest

## 프로젝트 실행 방법

### Install

```bash
npm install
```

### Start

```bash
npm run start
```

## 소요기간
2023.04.23 ~ 2023.04.24

## 사용 기술

- React
- Recoil
- typescript
- vite
